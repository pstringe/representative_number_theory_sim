## Function: `f_isalpha`

cCopy code

`int f_isalpha(int c);`

**Description:**

This function checks if the given character is an alphabetic character.

**Parameters:**

- `c`: Character to be checked.

**Returns:**

- `1` if the character is alphabetic.
- `0` otherwise.
