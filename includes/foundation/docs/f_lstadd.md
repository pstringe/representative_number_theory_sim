## Function: `f_lstadd`

cCopy code

`void f_lstadd(t_list **alst, t_list *new);`

**Description:**

This function adds a new node `new` at the beginning of the linked list `alst`.

**Parameters:**

- `alst`: Pointer to the pointer to the linked list.
- `new`: Pointer to the node to be added.

**Returns:**

None.
