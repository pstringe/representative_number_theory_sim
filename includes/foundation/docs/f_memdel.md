## Function: `f_memdel`

cCopy code

`void f_memdel(void **ap);`

**Description:**

This function frees the memory pointed to by `ap` and sets the pointer to `NULL`.

**Parameters:**

- `ap`: Pointer to the memory to be freed.

**Returns:**

None.
