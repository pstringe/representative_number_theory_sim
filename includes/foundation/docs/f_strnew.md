## Function: `f_strnew`

cCopy code

`char *f_strnew(size_t size);`

**Description:**

This function allocates a new string of the given `size` and initializes it to zero.

**Parameters:**

- `size`: Size of the string to allocate.

**Returns:**

Pointer to the allocated string.
