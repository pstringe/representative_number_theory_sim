## Function: `f_strdel`

cCopy code

`char *f_strdel(char **as);`

**Description:**

This function frees the memory allocated for a string and sets the pointer to `NULL`.

**Parameters:**

- `as`: Pointer to the string to be freed.

**Returns:**

Pointer to `NULL`.
