## Function: `f_isascii`

cCopy code

`int f_isascii(int c);`

**Description:**

This function checks if the given character is an ASCII character.

**Parameters:**

- `c`: Character to be checked.

**Returns:**

- `1` if the character is an ASCII character.
- `0` otherwise.
