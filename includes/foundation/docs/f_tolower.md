## Function: `f_tolower`

cCopy code

`int f_tolower(int c);`

**Description:**

This function converts the given character to lowercase.

**Parameters:**

- `c`: Character to be converted.

**Returns:**

The lowercase equivalent of the character if it is an uppercase character; otherwise, it returns the same character.
