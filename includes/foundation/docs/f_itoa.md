## Function: `f_itoa`

cCopy code

`char *f_itoa(int n);`

**Description:**

This function converts the integer `n` to a string.

**Parameters:**

- `n`: Integer to be converted.

**Returns:**

Pointer to the newly allocated string representation of `n`.
