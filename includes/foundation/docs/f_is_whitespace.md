## Function: `f_iswhitespace`

cCopy code

`int f_iswhitespace(int c);`

**Description:**

This function checks if the given character is a whitespace character.

**Parameters:**

- `c`: Character to be checked.

**Returns:**

- `1` if the character is a whitespace character.
- `0` otherwise.
