## Function: `f_lstiter`

cCopy code

`void f_lstiter(t_list *lst, void (*f)(t_list *elem));`

**Description:**

This function applies the function `f` to each node of the linked list `lst`.

**Parameters:**

- `lst`: Pointer to the linked list.
- `f`: Pointer to the function to be applied to each node.

**Returns:**

None.
