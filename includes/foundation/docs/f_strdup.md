## Function: `f_strdup`

`char *f_strdup(const char *str);`

**Description:**

This function creates a duplicate of the string `str`.

**Parameters:**

- `str`: Pointer to the string to be duplicated.

**Returns:**

Pointer to the newly allocated duplicated string.
