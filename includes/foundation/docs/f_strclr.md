## Function: `f_strclr`

cCopy code

`void f_strclr(char *s);`

**Description:**

This function sets every character of the string `s` to the null character (`'\0'`).

**Parameters:**

- `s`: Pointer to the string to be cleared.

**Returns:**

None.
