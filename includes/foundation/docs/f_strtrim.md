## Function: `f_strtrim`

`char *f_strtrim(char const *s);`

**Description:**

This function trims the leading and trailing whitespace characters from the string `s`.

**Parameters:**

- `s`: Pointer to the string to be trimmed.

**Returns:**

Pointer to the newly allocated trimmed string.
