## Function: `f_putstr_fd`

cCopy code

`void f_putstr_fd(char const *str, int fd);`

**Description:**

This function writes the string `str` to the specified file descriptor `fd`.

**Parameters:**

- `str`: Pointer to the string to be written.
- `fd`: File descriptor to write to.

**Returns:**

None.
