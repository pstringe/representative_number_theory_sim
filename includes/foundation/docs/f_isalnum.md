## Function: `f_isalnum`

`int f_isalnum(char c);`

**Description:**

This function checks if the given character is alphanumeric.

**Parameters:**

- `c`: Character to be checked.

**Returns:**

- `1` if the character is alphanumeric.
- `0` otherwise.
