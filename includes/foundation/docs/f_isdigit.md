## Function: `f_isdigit`

cCopy code

`int f_isdigit(char c);`

**Description:**

This function checks if the given character is a digit.

**Parameters:**

- `c`: Character to be checked.

**Returns:**

- `1` if the character is a digit.
- `0` otherwise.
