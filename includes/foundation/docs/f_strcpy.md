## Function: `f_strcpy`

`char *f_strcpy(char *dst, const char *src);`

**Description:**

This function copies the string `src` to `dst`, including the null-terminating character.

**Parameters:**

- `dst`: Pointer to the destination string.
- `src`: Pointer to the source string.

**Returns:**

Pointer
